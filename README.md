# Warpzone 2.0

This is a futuristic Next.js web application for Warpzone.

## Prerequisites

You need to have **Node.js** installed on your machine.
Download it from: [https://nodejs.org/](https://nodejs.org/)

## Getting Started

1.  **Install dependencies:**
    Open your terminal in this folder and run:
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```

3.  **Open the app:**
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Futuristic Design**: Neon colors, glassmorphism, and animations.
- **Tech Stack**: Next.js 14, Tailwind CSS, Lucide Icons.
- **Pages**: Home, Catalog, Services, Contact.
